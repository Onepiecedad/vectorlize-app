# Vectorlize
Image to SVG converter MVP.

## Monorepo
- frontend, React + TypeScript + Vite + Tailwind
- backend, Node + TypeScript + Express

## Run
### Backend
cd backend
cp .env.example .env
npm i
npm run dev

### Frontend
cd frontend
cp .env.example .env
# set VITE_API_URL to backend URL
npm i
npm run dev
